Self-hosted latin-subset woff2 (CSP `script-src 'self'` site; no external
font requests). All four are OFL 1.1 — redistribution in webfont form is
permitted; full license text: https://openfontlicense.org

| file | face | role | source |
|---|---|---|---|
| `ps2p.woff2` | Press Start 2P (CodeMan38) | arcade display — wordmark, verdict chrome, section titles | Google Fonts v16 |
| `dotgothic16.woff2` | DotGothic16 (Fontworks) | JRPG menu face — numerals, labels, window text | Google Fonts v21 |
| `jbmono.woff2` | JetBrains Mono (JetBrains, variable wght 100–800) | data/hashes/tables | Google Fonts v24 |
| `oracle-italic.woff2` | Cormorant Garamond Italic (C. Sorkin) | oracle prose only | Google Fonts v21 |

Rule: these four are the whole set. New faces need a reason and this note
updated. `@font-face` lives in `deck-ui.css`; families are named ONLY via
tokens in `deck-tokens.css`.
